This plugin is the Parent plugin of "childpublicpost" .Simple install this plugin on you site (main site).

== Description ==

The functionality of this plugin is if you want to showing your post on your another site so simple install "forpublicpost" plugin on your site and install "childpublicpost" on your sub site.
Add your sub site credentails.
So soap service fetching your post from your parent site and showing you sub site. 
If admin approve post showing for sub site.    

== Installation ==

1. Download `forpublicpost.zip` and upload contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you are showing a menu in sidebar


How to use plugin

1. Now go to "New Credantial" and add your credantial. (sub site details).
2. Then go to "For Public post menu" and select which post you want to showing on child site.
3. Then if you select all the post then click on "Set for Public " button then all the post send on you child site and store it the check it on child site.
